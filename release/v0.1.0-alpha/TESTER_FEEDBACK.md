## To test Smart Fix

1. Open a real code project (any repo with git)
2. Open Sompter → select the project in the dropdown
3. Click **🔧 Smart Fix**
4. Wait for it to read the screen, build a task, and run
5. Check review output

## To test Browser Control

1. Click the footer mode until it shows **Browser** (not OS)
2. Click **Plan Action** in the control panel
3. Approve the suggested browser action
